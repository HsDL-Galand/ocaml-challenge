COMANDI DI GIT
git status		(?) controlla lo status della repository
git add	<file>	(?) aggiunge i file alla lista dei file da pushare/committare
git commit		(?) committa in automatico tutti i file addati
	-am			"a": tutti i file, "m": messaggio
git push		(?) pusha nella repository 
git log:		per assicurarsi che sia andato tutto bene
